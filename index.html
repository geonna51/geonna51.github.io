Projects

NBA Trade Analyzer

The NBA Trade Analyzer is a web application built to analyze and evaluate potential trades in the NBA. The application allows users to input trade details and evaluates the fairness and impact of the trade using custom algorithms.

  • Technologies Used: Flask, Python, HTML, CSS
  • SQL database for trade data storage and retrieval
  • Bootstrap for responsive UI design
  • JavaScript for dynamic elements

The web app features include player comparisons, team analysis, and the ability to simulate trades across multiple teams. It offers users a seamless interface to evaluate whether proposed trades are realistic and beneficial for the teams involved.
